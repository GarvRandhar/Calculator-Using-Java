# Calculator Using Java

A simple calculator application built using Java. This project demonstrates basic arithmetic operations and GUI implementation in Java.

## Features

- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- User-friendly graphical interface
- Clean and simple design
- Error handling for invalid operations

## Technologies Used

- **Java** - 100%

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans) or a text editor

### Installation

1. Clone the repository
```bash
git clone https://github.com/GarvRandhar/Calculator-Using-Java.git
```

2. Navigate to the project directory
```bash
cd Calculator-Using-Java
```

3. Compile the Java files
```bash
javac *.java
```

4. Run the application
```bash
java Calculator
```

## Usage

1. Launch the calculator application
2. Enter numbers using the number buttons or keyboard
3. Select the desired operation (+, -, *, /)
4. Press the equals button to see the result
5. Use the clear button to reset the calculator
